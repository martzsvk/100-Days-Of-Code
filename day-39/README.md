# Day 39 & 40: Flight Deal Finder (Capstone) ✈️

A sophisticated travel hacking tool that monitors flight prices from your home city to various global destinations. If a deal is found that is lower than your target price, the system alerts you immediately.

## API Note: Amadeus & Maintenance ⚠️
As of my current development, the Amadeus API has undergone significant changes/deprecation for certain flight search endpoints. This project is built to demonstrate the logic of flight searching and data management.
