# Day 11: Blackjack Project 🃏

Simplified version of blackjack built in Python console. This project was the first major challenge as I had to combine everything I learned in the first 10 days.

## What I Learned 🛠️
* **Game State Management:** Tracking cards and scores for both the player and the dealer (computer) simultaneously.
* **Code Refactoring:** Moving repetitive logic into functions to keep the main game loop readable.
* **Problem Decomposition:** Breaking down a multi-step game into small, testable functions.

## How to Run 🚀
1. Run `main.py`
2. Type "yes" to play
3. Decide to get another card or pass
4. Try to win
