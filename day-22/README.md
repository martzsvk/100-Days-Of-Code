# Day 22: The Classic Pong Game 🏓

A 2-player version of the legendary arcade game Pong. This project focuses on managing multiple class instances, detecting collisions with moving objects, and creating a real-time scoring system.

## What I Learned 🛠️
* **Movement Logic:** Implementing the ball's diagonal movement.
* **Collision Math:** Calculating the distance between the paddle and the ball to trigger a bounce, and detecting when the ball hits the top/bottom walls.
* **Dynamic Difficulty:** Increasing the ball's speed every time it hits a paddle to make the game more challenging over time.

## Controls
* Player 1 (Left): `W` and `S` keys
* Player 2 (Right): `Up` and `Down` arrows
* You need to end the game manually or it will go into eternity :D
