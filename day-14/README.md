# Day 14: Higher Lower Game 📊

### The final project of the "Beginner" section!!! 
This game challenges the player to guess which Instagram account has more followers. It's a comprehensive exercise in data manipulation and game state management.

## What I Learned 🛠️
* **Working with Real Data:** Using a large dataset (list of dictionaries) to pull information about celebrities and brands.
* **Game Loop Logic:** Creating a continuous experience where the "Winner" (Option B) becomes the next "Contender" (Option A) in the following round.
* **Complex Conditionals:** Comparing follower counts and updating the score or ending the game based on the player's choice.

## How to Run 🚀
1. Run `main.py`
2. Compare the descriptions (Name, profession, country).
3. Type 'A' or 'B' and try to beat your high score!
