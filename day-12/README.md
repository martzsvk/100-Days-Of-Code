# Day 12: Number Guessing Game 🎯

A classic "Higher or Lower" number guessing game that demonstrates the use of Local vs. Global Scope and how to manage game constants.

## What I Learned 🛠️
* **Namespaces & Scope:** Understanding where variables can be accessed. Learned the difference between Local scope (inside functions) and Global scope.
* **Global Constants:** Practiced the convention of naming global constants in ALL_CAPS to make them easily identifiable.
* **Game Logic:** Implementing a difficulty system (Easy = 10 attempts, Hard = 5 attempts).

## How to Run 🚀
1. Run `main.py`
2. Choose a difficulty level: 'easy' or 'hard'.
3. Guess the number between 1 and 100.
4. The game will tell you if your guess is too high or too low until you win or run out of turns.
