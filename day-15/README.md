# Day 15/16: Coffee Machine (OOP Version) ☕

This project marks the transition from Procedural Programming to Object-Oriented Programming (OOP). Instead of using a single file with many functions, I built a modular system that simulates a real-world coffee vending machine.

## What I Learned 🛠️
* **Classes and Objects:** Creating Classes for the `CoffeeMaker`, `MoneyMachine`, and `Menu` to handle different responsibilities.
* **Encapsulation:** Hiding the internal logic of how coffee is made or how money is processed inside methods, making the main code much cleaner.
* **Real-world Logic:** Handling resource management (water, milk, coffee) and financial transactions (inserting coins, checking if enough money was provided).

## How to Run 🚀
1. Run `main.py`
2. Choose your drink: espresso / latte / cappuccino
3. Use the special commands:
   * `report` - See current resource levels and profit
   * `off` - Power down the machine
4. Insert coins (Quarters, Dimes, Nickels, Pennies)
5. Enjoy your virtual caffeine! ☕
