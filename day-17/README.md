# Day 17: The Quiz Project ❓

A true/false quiz game that uses Object-Oriented Programming to manage questions, scores, and game flow. This project focuses on creating custom classes.

## What I Learned 🛠️
* **Attributes vs. Methods:** Understanding how to store data in attributes and create functionality using methods.
* **The Quiz Brain:** Developing a `QuizBrain` class to manage the logic: tracking the current question number, checking answers, and determining if the quiz has ended.

## How to Run 🚀
1. Run `main.py`
2. Answer the questions by typing 'True' or 'False'
3. See your final score at the end of the round
