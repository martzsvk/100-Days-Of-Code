# Day 9: Secret Auction Program 💰

A private bidding console application where multiple users can enter their names and bid amounts. The program keeps all bids secret until the end, then automatically calculates and announces the winner.

## What I Learned 🛠️
* **Python Dictionaries:** Learned how to create, edit and retrieve data from dictionaries using keys and values.
* **Revision of While Loops**

## How to Run 🚀
1. Copy the code
2. Run `main.py` in your IDE/Terminal
3. Bid and see who wins
